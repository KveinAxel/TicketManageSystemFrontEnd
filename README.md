# 机票管理系统前端

本项目为2020吉林大学卓班数据结构课程设计

本项目为node.js程序设计

## 如何使用

1. 安装node.js
2. `git clone git@github.com:KveinAxel/TicketManageSystemFrontEnd.git`
3. `cd TicketManageSystemFrontEnd\`
4. `npm install`
5. `npm run dev`
